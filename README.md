This project demonstrates an AI-based Network Intrusion Detection System (NIDS) using Machine Learning (Random Forest) to detect DDoS attacks in network traffic.
It also integrates Generative AI (Grok) to explain why a network packet is classified as malicious or benign.
The model is trained on a real-world dataset from CIC-IDS2017 and helps simulate how AI can assist cybersecurity analysis.
